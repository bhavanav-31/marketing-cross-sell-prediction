# marketing-cross-sell-prediction
Predicting which existing insurance customers are likely to respond to a vehicle insurance cross-sell campaign using machine learning classification models.

## Problem

Insurance companies want to identify existing customers who are most likely to purchase vehicle insurance. Predicting these customers helps improve marketing efficiency, increase conversion rates, and reduce campaign costs.

## Dataset

- train_split.csv – Training dataset used to build the machine learning models.
- test_split.csv – Test dataset used to evaluate model performance.

## Models Compared

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Results

The models were evaluated using classification metrics such as Accuracy, Precision, Recall, F1-Score, and ROC-AUC.

- Random Forest achieved the highest F1-Score.
- Gradient Boosting achieved the highest ROC-AUC.
